# 셀폼 CEO 리뷰 요약

> 일자: 2026-06-23  
> 모드: 선택적 보강 — 최종 제품 범위 유지

## 검토 결론

셀폼은 소싱 상품을 신뢰 가능한 판매 콘텐츠로 바꾸는 AI 상품 콘텐츠 스튜디오로 유지한다. 커머스 운영 OS로 범위를 넓히지 않는다.

## 확정된 보강 사항

1. 초안은 미확인 사실을 경고와 함께 표시할 수 있으나, 최종 판매처 출력은 확인된 사실만 사용한다.
2. 공급처 링크 추출 실패는 사진·텍스트·옵션표 수동 입력으로 무중단 전환한다.
3. 외부 링크와 업로드 파일에는 서버 측 안전 검사를 적용한다.
4. 자동 저장, 중복 방지, 새로고침 후 작업 재개, 오래된 AI 결과의 덮어쓰기 방지를 적용한다.
5. 자료 수집·사실·카테고리·AI 생성·레이아웃·검수·출력을 독립 모듈로 분리한다.
6. 카테고리별 상품 테스트 팩과 출력 스크린샷 비교를 유지한다.
7. 프로젝트별 AI·렌더링 작업 추적, 비용·실패 관측, 오류 알림을 적용한다.
8. 사실·페이지·출력물을 버전과 근거로 고정해 재현 가능하게 한다.
9. API는 리소스 중심, 비동기 작업 ID, 중복 없는 요청으로 설계한다.
10. 섹션 단위 재생성, 이미지 최적화, 작업량·사용량 제한을 적용한다.
11. 편집기는 가이드형 3단 구조와 행동 가능한 검수 경험을 기본으로 한다.

## 명시적 제외 범위

- 1.0에서 판매처 자동 등록, 자유 배치 캔버스, 자체 결제, 팀 협업·구독, 고위험 생성 이미지 편집, 성과 기반 자동 최적화는 제외한다.
