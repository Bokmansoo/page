## 3. Issues and Next Sprint Candidates
테스트 과정에서 발견된 주요 UX 특이사항 및 향후 스프린트 개선 후보를 정리합니다.

1. **상세페이지 에디터와 사실 확인 페이지 간 이동 편의성 (Minor)**:
   - 사실 카드가 부족할 때 에디터나 스타일 선택 단계에서 사실 확인 페이지로 직접 되돌아가서 사실을 추가할 수 있는 바로가기 링크 버튼이 추가되면 사용자 경험이 더 매끄러울 것입니다.
2. **섹션 텍스트 자동 채우기/AI 추천 기능 (Major)**:
   - 20자 미만 경고 노출 시, 사용자가 직접 쓰는 것 외에도 "AI가 추천 문장 제안하기" 버튼을 배치하여 원클릭 보정이 가능하면 문구 품질 관리가 훨씬 쉬워질 것입니다.
