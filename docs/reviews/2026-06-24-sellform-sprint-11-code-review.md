## 1. 변경 요약
- **로딩 및 빈 상태 텍스트 개선**: 유저 친화적인 표현으로 변경하였고, 상품 사실 카드가 없는 빈 화면에서도 `AI로 사실 카드 자동 생성`과 `사실 카드 수동 추가` 액션 버튼을 즉각 찾을 수 있도록 레이아웃을 고도화했습니다.
- **감사 모달 오기 교정**: `"사실 카드 변경 이력 (감사 감사)"`로 되어있던 텍스트 오기를 `"사실 카드 변경 이력 (감사 로그)"`로 올바르게 수정했습니다.
- **데이터 메타데이터의 시각적 한글 맵핑**: 
  - `extraction_source`를 원문 키(`manual_text`, `image` 등) 대신 `"텍스트 근거"`, `"이미지 근거"` 등으로 직관적으로 맵핑하여 배지를 노출했습니다.
  - 신뢰도를 `신뢰도 N%` 형식으로 포맷팅했습니다.
  - `risk_flags`를 `"인증/표기 확인 필요"`, `"낮은 신뢰도"` 등으로 한글화하여 법률/인증 규정 검수가 용이해졌습니다.
- **모바일 웹 반응형 정렬 보완**: 상단 헤더 액션 그룹의 Flex Wrap 처리를 보완하여 모바일 폭(390px 이하)에서 버튼들이 밀리거나 깨지지 않고 예쁘게 떨어지도록 조치했습니다.
