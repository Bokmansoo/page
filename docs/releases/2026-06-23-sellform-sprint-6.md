# 릴리스 노트: Sellform Sprint 6

## 추가된 기능

- 상품 상세페이지를 모바일 웹형 랜딩페이지로 발행할 수 있다.
- 공개 URL 또는 slug로 비인증 고객이 페이지를 볼 수 있다.
- 공개/비공개 전환으로 이미 공유된 페이지 접근을 차단할 수 있다.
- 구매 버튼을 쿠팡/스마트스토어 등 외부 판매 링크에 연결할 수 있다.
- 공개 페이지에 이미지 갤러리, FAQ 아코디언, Before/After 슬라이더, 영상 임베딩을 표시할 수 있다.

## 개선된 점

- 공개 페이지와 발행 관리 화면의 깨진 한국어 문구를 정리했다.
- 이미지가 없는 상품은 텍스트 중심 fallback으로 표시된다.
- 구매 링크가 없으면 구매 버튼이 비활성 상태로 표시된다.
