# Sellform 브라우저 보조 상품 정보 수집 가이드

본 가이드는 쇼핑몰 등의 자동 크롤링 방지 정책(로그인 요구, 캡차 등)으로 인해 상품 링크 수집에 실패했을 때, 사용자의 브라우저를 통해 안전하고 신속하게 상품 상세 정보를 수집하고 이를 Sellform의 사실 카드 후보로 매핑하는 절차를 설명합니다.

---

## 1. 수집 프로세스 단계별 가이드

### 1단계: 상품 링크 입력
- 새 프로젝트 생성 페이지에서 수집하려는 상품 공급처 URL(예: 타오바오, 1688, 쿠팡, 스마트스토어 등)을 입력하고 `분석 및 수집` 버튼을 누릅니다.

### 2단계: AI 자동 수집 시도 및 Fallback 감지
- 서버에서 1차 HTTP 직접 수집 및 2차 웹 검색 보조 수집을 시도합니다.
- 만약 보안 접근 제어 필터에 막혀 수집 실패 메시지가 나타나면, 수동 자료 입력 또는 사실 확인 검증 보드의 **브라우저 보조 정보 수집기** 패널로 안내를 전환합니다.

### 3단계: 상품 페이지 새 탭에서 열기
- 안내 패널 상단의 `💻 상품 페이지 열기 ↗` 버튼을 클릭하여, 공급처의 원본 상품 상세페이지를 브라우저 새 탭으로 띄웁니다.

### 4단계: 상품 스펙 및 상세 설명 텍스트 복사 (Copy)
- 열린 상품 페이지에서 상세 설명 텍스트, 구성품 정보, 규격 테이블의 텍스트 등을 마우스 드래그를 사용하여 전체 또는 일부 영역을 복사(Ctrl + C)합니다.

### 5단계: Sellform 브라우저 보조 수집 패널에 붙여넣기 (Paste)
- 복사한 내용을 Sellform의 `1. 상품 상세 정보 / 스펙 설명 복사 후 붙여넣기` 텍스트 상자(Textarea)에 붙여넣습니다.

### 6단계: 여러 사실로 변환 (Bulk Parse)
- `⚙️ 여러 사실 후보로 변환하기` 버튼을 누릅니다. 
- 입력된 텍스트 중 줄바꿈 및 특수 기호를 필터링하여 최소 3자 이상의 온전한 텍스트 행들을 개별 사실 후보 카드로 신속히 파싱합니다.
- 오른쪽 미리보기 영역에서 추출된 각 사실 후보를 검토하고, 불필요한 행이 있다면 개별 `✕` 버튼을 눌러 제외합니다.

### 7단계: 후보 사실 카드로 저장
- `📥 사실 검증 대기 후보로 저장하기` 버튼을 클릭하여 일괄 등록합니다.
- 등록된 카드들은 즉시 사실 검증 테이블에 추가(Verification Status: `unknown` 상태)됩니다.

### 8단계: 후보 검수 및 확인됨 처리
- 수집된 사실 카드를 하나씩 보며 오류나 과장 문구가 없는지 검수하고, 신뢰할 수 있는 사실에 대해 `확인됨 (Confirmed)`으로 체크하여 상세페이지 생성 자격을 획득합니다. (최소 3개 이상 필요)

---

## 2. 수집 및 정책 주의사항 (Safety Policy)

> [!IMPORTANT]
> **우회 수집을 시도하지 않습니다.**
> Sellform은 로그인 세션 탈취, 캡차 무력화, 비정상 대량 트래픽 크롤링 등의 크래킹 공격이나 우회 동작을 수행하지 않습니다. 이는 공급처 사이트의 보안 정책 및 법적 분쟁을 방지하기 위한 안전 장치입니다.

> [!TIP]
> **사용자가 직접 볼 수 있는 정보만 근거로 활용합니다.**
> 브라우저 보조 수집은 사용자가 직접 로그인하여 접근 가능한 페이지에서 정상적으로 드래그 복사한 텍스트만을 투명하게 시스템의 근거로 사용하므로 보안과 정책 면에서 안전합니다.

> [!WARNING]
> **근거가 없는 내용은 상세페이지 카피로 추천하지 않습니다.**
> 팩트 검증 카드에 등록되어 `confirmed` 상태로 지정된 명확한 사실 기반 데이터만을 AI가 조립하므로, 근거 없는 허위 과장 광고 문구의 상세페이지 생성을 사전에 완벽히 방어합니다.
